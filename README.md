
# UBike 
 使用Java語言、Room資料庫、Google Maps API,  
 實作出單車租借即時動態查詢`地圖導航Android APP。
 
 功能:
•提供雙北UBike站點資訊。  
•提供雙北UBike租借即時查詢 (可借車輛/可停空位)。  
•提供我的最愛功能。  
•提供Google推薦步行路線。
